    使用jdbc连接sqlserver数据库以及连接gbase数据库

    工具 工具
    创建时间:2019-05-16 13:26
    阅读:
    评论:

    最近的项目有个需求是需要连接到10个数据库，从其中的9个库查询数据然后插入到另一个数据，有sqlserver数据源和gbase数据源，记录一下连接方式；

    sqlserver：

    Class.forName(“net.sourceforge.jtds.jdbc.Driver”);

    不带实例名称

    Connection conn = DriverManager.getConnection(“jdbc:jtds:sqlserver://xxxx:1433;DatabaseName=xx;user=xx;password=xx”);

    带实例名称

    Connection conn = DriverManager.getConnection(“jdbc:jtds:sqlserver://xxxx;instance=xx;DatabaseName=xx;user=xx;password=xx”);

    gbase:

    Connection conn = DriverManager.getConnection(“jdbc:gbase://xxx:5258/库名”,”账号”，“密码”);
